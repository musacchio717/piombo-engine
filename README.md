# Piombo Engine

Gioco testuale d'avventura distopico con AI narrator.
Inference locale su RTX 4070, zero cloud dependencies.

## Stack
- Backend: Python 3.12 + FastAPI
- IA: LangGraph + Qwen3-8B (Ollama)
- Knowledge Graph: Neo4j + NetworkX
- Vector Store: Qdrant
- Frontend: React 18 + Vite
